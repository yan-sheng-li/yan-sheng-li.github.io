## 安装

```shell
yarn add vuepress-plugin-reading-progress
# or
npm i vuepress-plugin-reading-progress
```
## 使用
> 在`config.js`配置文件里：
```js
module.exports = {
  plugins: [
    'reading-progress'
  ]
}
```
