## 项目背景 

学籍信息管理是学校日常工作中的一项重要任务。传统的纸质档案管理方式已经无法满足信息化时代的需求。因此，我们开发了这个学籍信息管理系统，旨在提高学校信息管理效率，简化工作流程。

## 技术选择

- Java
- MySQL
- JDBC
- SwingGUI

## 系统功能

- 学生信息增删改查
- 多条件模糊查询
- 嘉奖
- 加学分
- 学分实时排名
- 学生个人获奖详情
- 多条件检索
- 可继续扩展
