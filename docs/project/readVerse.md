# 📕在线小说网 <Badge type="warning" text="毕设" vertical="top" />

<MyGlobalComponent />

## 技术栈
- SpringBoot
- MyBatisPlus
- MySQL数据库
- BootStrap
- Jquery
- sa-token

## 要点
- 用户登录注册
- 验证码校验
- 小说列表，分类，推荐
- 小说打赏
- 小说收藏
- 小说章节
- 小说阅读，上下章节调整，字号调整
- 用户信息维护
- 分类检索，全局检索
- 支持发布小说
- 用户管理
- 小说管理
- 评论管理
- 等等....（可继续扩展）
