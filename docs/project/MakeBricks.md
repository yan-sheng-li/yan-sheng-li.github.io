## 要点：

- 左右方向键控制挡板移动
- 回车键重新开始
- 球撞到底部游戏失败，砖块全部清除游戏胜利
- 可自定义游戏界面背景、球、挡板贴图
- 支持多关卡切换，砖块消完后，按回车可进入下一关，通关后可重新开始。
- 支持一次多球，只要球场还有球在，游戏就继续
- 砖块被击碎时有概率掉落道具（加速球、减速球、加长/缩短挡板）
