## 技术栈
- 开发语言：Java
- 数据库：MySQL
- 依赖：JDBC
- 交互：Swing
- 编码：GBK

## 要点
- 登录认证
- 进货记录
- 售货自动结算
- 多条件信息查询（库存、进货、售货）
