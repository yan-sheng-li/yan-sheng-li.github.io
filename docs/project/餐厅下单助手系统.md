## 技术栈

- Java
- SwingUI
- JDBC
- MySQL

## 要点提炼

- 分角色登录，顾客可注册
- 商家端可菜品管理，订单管理，用户管理
- 顾客端筛选菜品，加购，购物车菜品数量调整
- 下单金额计算，下单确认
- 订单信息汇总、详情
- 均可个人信息修改，登录密码修改
- 还可继续扩展
