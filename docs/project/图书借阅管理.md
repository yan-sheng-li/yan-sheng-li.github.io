# 📘图书借阅管理

## 技术栈

- Java
- MySQL
- Swing
- JDBC

## 功能要点
- 登录验证
- 借书证办理、变更、挂失、注销、查找
- 读者类型维护
- 图书管理、入库
- 简单检索，高级检索
- 图书借出、归还
- 图书续借
- 逾期、超期金额生成
- 等等
