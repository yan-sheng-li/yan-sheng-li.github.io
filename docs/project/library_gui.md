# 📗图书资源管理系统*

<MyGlobalComponent />

## 技术栈
- Java语言
- MySQL数据库
- JDBC
- Swing GUI

## 要点
- 用户登录，分管理和读者角色
- 图书检索方式多样，任意检索，简单检索，组合检索
- 用户可借书，还书，预约借书，个人借阅历史
- 管理员可管理图书信息，用户信息
- 管理员可查看读书借阅情况，数据汇总统计，图表分析
- 表格重点数据颜色标记
