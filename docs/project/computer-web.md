# 电脑销售网站（Jsp）

<MyGlobalComponent />

<VersionSwitcher>

## 技术栈

- Servlet+Jsp
- JDBC
- MySQL
- c3p0
- commons-dbutils


## 要点

- 双角色：管理员、买家
- 用户登录
- 新用户注册
- 商品选购
- 购物车
- 订单信息
- 用户管理
- 商品管理
- 订单管理
