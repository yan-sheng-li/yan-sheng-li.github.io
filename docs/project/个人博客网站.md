## 技术栈
- SpringBoot
- MySQL
- Thymeleaf
- MyBatis

## 功能要点
- 博客首页

- 文章分类

- 文章标签

- 发布时间线

- 作者简介页

- 文章检索

- 文章支持markdown格式

- 数据分页，分类索引

- 页面动漫人物互动

- 后台登录

- 文章，标签，分类管理

- 内容统计

- 网页背景可调节

- 可继续扩展更多的功能
