## 技术栈
- SpringBoot
- MySQL
- MybatisPlus
- LayUI
- SaToken
- Thymeleaf

## 要点
> 前台
- 首页资讯
- 旅游路线搜索，发布，收藏
- 在线聊天室
- 团游详情
- 旅游组团
- 游记查看，发布，评论，收藏
- 收藏列表
- 个人信息维护
- 等等

> 后台
- 路线管理
- 游记管理
- 用户管理
- 评论管理
- 团游管理
