# 📆电子日历记事本

<MyGlobalComponent />

## 技术栈
- Java语言
- SwingUI
- IDEA

## 要点
- 菜单栏中提供编辑（剪切、复制、粘贴）、保存、打开等功能。
- 使用类组件实现图形界面设计。
- 基于图形界面的日历，用户可编辑或查看指定日期的日志内容。
- 提供按钮实现月份的前后翻动。
- 事件持久化到文件，可再次编辑保存
