# 电脑销售商城（Vue）

<MyGlobalComponent />


<VersionSwitcher>

## 技术栈

- SpringBoot
- Vue
- MySQL
- Myabtis

### 要点

> 用户端

- 登录认证
- 新用户注册
- 商品多方式筛选
- 商品收藏
- 个人信息
- 购物车下单
- 订单处理

> 管理员端

- 商品管理
- 分类管理
- 订单管理
- 用户管理
- 数据统计
- 商品信息上传
