# 🏓运动会信息管理系统

<MyGlobalComponent />

> 本课题主要实现对运动会信息的管理，基础功能包含运动员报名比赛，比赛成绩查询，资讯留言等功能，可扩展性较高，使用主流的开发技术实现，项目结构规范。

## 技术选型
- MySQL
- Springboot
- Sa-token
- Mybatis-plus
- Thymeleaf
- LayUI
