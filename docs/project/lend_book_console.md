# 📚图书借阅管理系统

<MyGlobalComponent />

## 需求分析

> 1. 系统分为管理员账户，学生（读者）账户
> 2. 管理员账户可以进行书籍，学生（读者）信息的维护(CRUD)
> 3. 学生（读者）账户可以借书，还书，检索图书，个人借阅信息浏览
## 开发环境

`开发语言`：Java   (jdk版本12+)

`数据库`：MySQL  (版本8.0+)

`工具`：JDBC

`交互场景`：控制台

`软件`：IDEA（推荐），Navicat

`项目构建`：maven
