# 🚘二手车交易管理*

<MyGlobalComponent />

## 技术栈
- Java
- MySQL
- JDBC
- Swing GUI

## 要点
- 买家/卖家注册、登录
- 买家可浏览车产品，下订单
- 卖家可管理车产品，处理订单
- 个人信息维护
- 可批量导入数据
- 可继续扩展
