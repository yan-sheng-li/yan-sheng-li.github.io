# 📒微信与支付宝的收款码二合一

## 背景
很多小伙伴在日常使用个人微信、支付宝收款码收款时，都想使用工具把自己的微信支付收款码和支付宝收款码二码合一，但市面上的合码工具质量参差不齐，要么是收费的、要么是免费但合码质量效果差，而市面上大部分的合码工具都是通过跳转链接后出现最终的收款码，付款方要长按识别后才能进行付款，严重影响收款体验
