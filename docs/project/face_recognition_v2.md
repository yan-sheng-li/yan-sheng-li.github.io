<MyGlobalComponent />

## 技术栈

- Python
- OpenCV
- face_recognition
- Tkinter
- Pillow
- Numpy

## 要点

- 基于 Python 的人脸识别应用
- 支持 **实时摄像头识别** 与 **图片对比识别**
- Tkinter 构建简洁 GUI 界面
- 集成 Pillow 解决中文显示问题
- 自动识别人名或标记为“未知”
- 人脸库管理
- 人脸信息采集
- 增强识别精度

## 功能模块

1. **实时人脸识别**
   - 调用摄像头实时检测与标注人脸
   - 识别结果实时显示在界面中
2. **图片人脸对比**
   - 选择两张本地图片进行特征比对
   - 判断是否为同一人并输出结果
3. **人脸录入**
   - 摄像头采集
   - 画面保存

4. **人脸管理**
   - 支持人脸信息的增删改查
   - 支持图片替换
